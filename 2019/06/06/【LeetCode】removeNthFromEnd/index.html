【LeetCode】removeNthFromEnd

  • LeetCode

删除倒数第 n 个节点

思路 1:

  1. 循环第一次找出链表长度;
  2. 计算长度-n 得出需要删除的节点位置;
  3. 根据节点的位置循环找到需要删除的节点的前一个节点;
  4. 将该节点的指向改为 head.next.next(即删除节点后一个节点);

思路一循环次数过多，容易出现错误,对特殊的链表处理不当会出现 bug

思路 2：

  1. 将链表依次保存到一个 list 集合中(集合编号 0 设置为头结点的指向);
  2. 集合长度-n 德都需要删除的节点的下标;
  3. 根据下标找到前一个节点，将其指向删除节点所指向的节点;

思路二利用集合减少循环此次很好的保存每个节点
