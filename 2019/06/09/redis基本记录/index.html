redis基本记录

  • 随笔

记录一下Redis的一些学习

  1. Redis看了些基础发现比较容易理解基本操作都是围绕set与get两个指令进行

  2. set有 set hset hmset 还有个特殊的lpush,感觉很有意思比较容易理解,

  • set不用说与java中的set一样设置或者说添加，set key1 1 set key1 “11111”

  • hset 第一个h应该是hash哈希表的意思 也就是一个添加hash表的指令；hset key2 username adminn

  • hmset 也是hash表操作 中间的m不知道啥意思 可以以 <key> <val> 的方式循环赋值下去（应该有上限吧- -） hmset key6 username admin age 16

  • lpush l应该是左的意思向左添加 rpush同理 push以前在java与汇编中都看到过和set差不多 lpush key3 a rpush key3 b;

  1. get有 get hget hmget hgetall 以及与lpush对应的lrange

  • get hget hmget 和上面几个对应都是获取值 get key1 hget key2 username （单个） hmget key6 username age （多个）

  • lrange 输出列表从左输出lrange key3 <起始位置><结束位置> 没有rrange,没有rrange,没有rrange,重要的事情说三遍 lrange key3 0 3

  • hgetall 通用的 一个输出全部的hash表

  1. keys * 查看全部的 key表 del key1 删除key1

记录的就这些，个人感觉命令都很短都是一些单词的简写，对英语不好的人很友好。

  • 工作中也用到了redis 但是好像就用到了一些注解，自动将结果存入redis中 做到缓存- -
